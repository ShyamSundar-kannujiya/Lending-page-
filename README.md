 SRMU Admission Registration Portal
 
A modern, responsive, and functional Registration Page clone for Shri Ramswaroop Memorial University (SRMU). This project demonstrates a clean UI/UX design with integrated form handling.

Overview
This project is a landing page designed to capture student registrations. It features a hero section with university branding and a structured registration form with real-time interactivity.

Key Features
* **Modern UI/UX:** High-quality background with an overlay for better text readability.
* **Responsive Layout:** Fully compatible with mobile, tablet, and desktop screens.
* **Interactive Form:** Includes input fields for Name, Email, Phone, and dynamic dropdowns for Program and Location.
* **Form Validation:** Built-in JavaScript to ensure data integrity before submission.
* **Authorization Checkbox:** Includes a mandatory consent checkbox for university communication.
 Tech Stack
* **HTML5:** Semantic structure of the webpage.
* **CSS3:** Custom styling, Flexbox layouts, and hover effects.
* **JavaScript:** Handling form interactions and basic validation logic.
